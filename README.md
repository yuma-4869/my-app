# my-app

初めての Pull Request を体験するための練習用リポジトリです。

## 概要

このプロジェクトには、簡単な挨拶を返す小さな関数が含まれています。
Git と GitHub を使った開発フロー（ブランチ作成 → コミット → プッシュ → PR）を
学ぶことを目的としています。

## 使い方

```powershell
node greet.js
```

実行すると、コンソールに挨拶メッセージが表示されます。

## テスト

依存パッケージは不要です。Node.js があれば次のコマンドで実行できます。

```powershell
node greet.test.js
```

すべて成功すると `✓ すべてのテストに合格しました` と表示されます。
